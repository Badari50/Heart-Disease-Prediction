# Heart-Disease-Prediction
Developed a machine learning model to predict heart disease risk using clinical data, incorporating data preprocessing, exploratory data analysis, and classification techniques to achieve accurate predictions.

# Dataset
The dataset contains medical attributes such as:
Age
Sex
Chest Pain Type
Blood Pressure
Cholesterol Levels
Heart Rate
Other clinical features

# Features used
Data preprocessing and cleaning
Exploratory Data Analysis (EDA)
Machine Learning model training
Prediction of heart disease risk
Simple and easy-to-use implementation

# Technologies Used
Python
Pandas
NumPy
Matplotlib / Seaborn
Scikit-learn
